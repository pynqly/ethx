Privacy Policy for ETH Yield Engine Last Updated: November 2025 ETH Yield Engine (“the Extension”, “we”, “us”) is committed to protecting your privacy. This Privacy Policy explains what data we do and do not collect when you use the Chrome Extension. 1. Information We Collect The Extension does NOT collect, store, or transmit any personally identifiable information (PII). We collect zero of the following: No name, email, or wallet address No browsing history No blockchain addresses No cookies No tracking identifiers No analytics No server-side logging of user activity 2. Local Storage The Extension may store non-sensitive settings in your browser’s chrome.storage (e.g., UI preferences, last selected network). This data: Never leaves your device Is not shared with any external server Can be deleted by uninstalling the extension 3. Data From GitHub The Extension fetches staking yield snapshots from a public GitHub file (snapshot.json). This request: Is read-only Contains no user data Is identical for all users Is not tracked or logged by us 4. Third-Party Services We do not use: Google Analytics Advertising frameworks Fingerprinting scripts Cookies No third-party receives any user data. 5. Security Because the Extension does not process personal data, the security risk to users is minimal. However, we still ensure: Secure HTTPS GitHub fetches No remote executable code No unnecessary permissions 6. Changes to This Policy We may update this Privacy Policy to reflect improvements or regulatory changes. Updates will be posted on the extension’s GitHub repository. 7.
